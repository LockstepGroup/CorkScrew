# Stop-CsLogger

## SYNOPSIS
Stops an instance of CsLogger.

## SYNTAX

```
Stop-CsLogger [<CommonParameters>]
```

## DESCRIPTION
Stops an instance of CsLogger.

## EXAMPLES

### Example 1
```powershell
>Stop-CsLogger
```

Stops active CsLogger instance.
