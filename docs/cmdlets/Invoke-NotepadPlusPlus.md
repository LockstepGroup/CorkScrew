# Invoke-NotepadPlusPlus

## SYNOPSIS
Just a quick shortcut to NotepadPlusPlus, alias npp.

## SYNTAX

```
Invoke-NotepadPlusPlus [[-Path] <String>] [<CommonParameters>]
```

## DESCRIPTION
Just a quick shortcut to NotepadPlusPlus, alias npp.

## EXAMPLES

### Example 1
```powershell
PS C:\> npp
```

Opens NotepadPlusPlus.

### Example 1
```powershell
PS C:\> npp todo.txt
```

Opens todo.txt file in NotepadPlusPlus.
